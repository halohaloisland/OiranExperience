# 鹿児島花魁体験 サイト管理マニュアル

このドキュメントでは、ウェブサイトのコンテンツ（ギャラリー、ニュース、プラン）を更新する方法について説明します。

## 1. ギャラリー画像の追加・更新

ギャラリーの画像は、フォルダに画像を置いてプログラムを実行するだけで、自動的にウェブサイトに反映されます。

### 手順

1.  **画像の準備**:
    *   追加したい画像を用意します（推奨：JPEGまたはPNG形式）。
    *   画像ファイル名は、英数字（例: `gallery_01.jpg`）にすることをお勧めします。

2.  **画像の保存**:
    *   以下のフォルダに画像を保存（コピー）してください。
    *   フォルダパス: `OiranExperience/assets/images/gallery/`
    *   *※不要になった画像はこのフォルダから削除してください。*

3.  **更新プログラムの実行**:
    *   ターミナル（コマンドプロンプト）を開き、`OiranExperience` フォルダに移動します。
    *   以下のコマンドを実行します。

    ```bash
    python3 update_gallery.py
    ```

    *   「Success! Updated 'gallery_data.js'...」と表示されれば完了です。
    *   ブラウザでページを更新（リロード）して、画像が反映されているか確認してください。

---

## 2. ニュース（お知らせ）の追加・更新

トップページの「News / お知らせ」セクションは、データファイル（`news_data.js`）を編集して更新します。
※ファイルを直接開くとエラーになる場合があるため、この方式に変更しました。

### 手順

1.  **ファイルの編集**:
    *   以下のファイルをテキストエディタ（メモ帳やVS Codeなど）で開きます。
    *   ファイルパス: `OiranExperience/news_data.js`

2.  **内容の記述**:
    *   `const newsData = [...]` という形式になっています。
    *   新しいニュースは一番上に追加します。

    ```javascript
    const newsData = [
        {
            "date": "2025.01.01",
            "title": "新年のご挨拶",
            "content": "明けましておめでとうございます。"
        }, // ← カンマを忘れずに！
        {
            "date": "2024.12.01",
            "title": "Winter Campaign Started",
            ...
        }
    ];
    ```

3.  **保存と確認**:
    *   ファイルを上書き保存します。
    *   ブラウザでページを更新して、ニュースが反映されているか確認してください。


---

## 3. プラン・価格の変更

プランや価格情報は `plans.html` ファイルに直接記述されています。

### 手順

1.  **ファイルの編集**:
    *   `OiranExperience/plans.html` をテキストエディタで開きます。

2.  **該当箇所の修正**:
    *   `<table class="pricing-table">` タグの中を探します。
    *   金額を変更したい場合は、例として `¥35,000` と書かれている部分を探して書き換えてください。

    ```html
    <td class="plan-price">¥35,000</td>
    ```

3.  **保存**:
    *   ファイルを上書き保存して、ブラウザで確認してください。

---

## 困ったときは

*   **画像が表示されない**: ファイル名に日本語やスペースが含まれていないか確認してください。また、`update_gallery.py` を実行したか確認してください。
*   **ニュースが消えた**: `news.json` のカンマ `,` の付け忘れや、括弧 `}` の閉じ忘れがないか確認してください。
